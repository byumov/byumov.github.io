Wireguard с UI

PUBLISHED ON DEC 10, 2023 / 0 MIN READ

Простой способ поднять wireguard с UI в одну команду:

export PASS="<password>"

podman run -d \
    --name=wg \
    -e WG_HOST=135.181.42.186 \
    -e PASSWORD=$PASS \
    -v ~/.wg-easy:/etc/wireguard \
    -p 51820:51820/udp \
    -p 51821:51821/tcp \
    --cap-add=NET_ADMIN \
    --cap-add=SYS_MODULE \
    --sysctl="net.ipv4.conf.all.src_valid_mark=1" \
    --sysctl="net.ipv4.ip_forward=1" \
    --restart unless-stopped \
    weejewel/wg-easy
